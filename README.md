# Discord-Stats-Bot

Show various stats for the last 30 days in a server.

This is made for English & Japanese servers.

## Testing

1. Set `DEBUG=true` in `.env`
2. Run the bot
   ```bash
   npm run dev
   ```

## Deploying

1. Install dependencies

   ```bash
   npm install
   ```

2. Create `.env` based on `sample.env`

3. Run the bot

   ```bash
   npm start
   ```
